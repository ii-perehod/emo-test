# ЭМО — Напечатай слово

Тренажёр для отработки навыка набора слов на клавиатуре. Пользователь видит картинку и вводит слово, которое она означает. Адресован людям с особенностями ментального развития, поэтому интерфейс крупный, контрастный, гласные подсвечены красным.

Тестовое задание для стажировки в АНО «Мастерская креативное ДЕЛО».

## Как запустить

Открыть `index.html` в браузере. Сборки и зависимостей нет.

## Структура

```
emo-test/
├── index.html       — экран с картинкой и полем ввода
├── answers.html     — экран итогов
├── styles.css       — стили
├── images/          — иллюстрации к заданиям
└── README.md
```

## Стек

HTML, CSS. Шрифты с Google Fonts: Inter и Palanquin Dark.

## Поддержка браузеров

Chrome, Firefox, Edge, Safari последних версий. Используются `container queries`, `transform`, CSS custom properties.

## Ключевые решения

**Адаптивность во весь экран.** Макет от Figma 1782×985. Фрейм тренажёра фиксирован на `100vw × 100vh` — заполняет окно при любом соотношении сторон, скролла нет. Внутренняя система координат двухосная: `--ux = 100cqi / 1782` для X, `--uy = 100cqb / 985` для Y. Позиции (`top`, `left`) и крупные контейнеры (карточки, колонки и строки таблицы) тянутся по своим осям независимо. Картинки, иконки, шрифты используют `--u = min(--ux, --uy)` — остаются пропорциональными при растяжении.

**Геометрия из Figma.** Все top/left/width/height/border-radius/цвета взяты из дизайна напрямую. Inner alignment учтена через `box-sizing: border-box`.

**Гласные.** Подсвечены `var(--color-text-vowel)` через `<span class="vowel">` — по требованию дизайна.

**SVG-спрайт.** На странице итогов галочка повторяется 4 раза, крестик 2 раза — вынесены в `<symbol>` + `<use>`, чтобы не дублировать длинные пути.

**Доступность.** Кнопки имеют `aria-label`, инпут — `<label>` (visually-hidden), декоративные SVG — `aria-hidden`, информационные — `role="img"` + `aria-label`.
